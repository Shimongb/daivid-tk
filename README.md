# daivid

Pre-trained Multi-layer Recurrent Neural Networks (LSTM, RNN) 
for analysis and generation of word class models in Python.



# Requirements
- [Tensorflow](http://www.tensorflow.org)

# Basic Usage

To get a sample text
```bash
python test.py
```
